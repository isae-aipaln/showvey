# Release Notes v2.0

기간: 2026년 7월 6일 ~ 7월 15일
단계: PC 지원 및 UI 전면 개선
상태: 배포 완료

# **[Showvey v2.0 릴리즈 노트]**

---

### **1. 개요 (Overview)**

모바일 전용이던 showvey가 **PC/노트북을 정식 지원**하는 플랫폼으로 확장된 메이저 업데이트입니다.
접속 기기에 따라 화면이 자동으로 전환되며, PC에서는 품평 후 유관부서 회의에서 스타일별 이미지·정보를 크게 보며 메모하는 활용이 가능해졌습니다.
아울러 이미지 비율 제약 해소, 이미지 확대 보기, 평가 저장 로직 강화 등 품평 경험 전반을 개선했습니다.

---

### **2. PC 버전 신규 지원 (Major)**

**① 기기별 자동 화면 전환**

- 화면 폭 1024px 기준으로 모바일/PC 레이아웃 자동 분기 (주소·계정 동일)
- **갤러리**: 8열 × 한 페이지 16개, 화면 전체 폭 활용, 세로 중앙 정렬
- **상품정보**: 좌측 이미지(화면 높이에 맞춰 자동 확대) + 우측 정보/평가 2단 구성, 이미지와 정보영역 세로 길이 일치, 하단 버튼은 이미지 하단 라인에 고정
- **코디평가**: 6열 그리드

**② 권한별 상품정보 페이지 최적 배치**

- 임직원1: [기본정보 → 상세정보(자동 펼침)] 스택 + 총평 열 — 상세정보가 총평과 하단 라인 일치
- 임직원2·매장: [기본정보 | 평가입력] 좌우 배치 + 아래 대형 총평(회의 메모용)

**③ PC↔모바일 화면 전환 버튼**

- 갤러리 우측 상단(공유 버튼 좌측)에서 원클릭 전환, 모든 권한 제공
- 모바일 화면 선택 시 폰 화면 그대로 중앙 표시, 선택 상태 유지(새로고침에도 보존)

### **3. 품평 UX 개선 (전 기기 공통)**

| 구분 | 내용 |
| --- | --- |
| **이미지 확대 팝업** | 상품정보·코디 이미지 클릭 시 중앙 확대 보기 — 배경 어둡게 처리, 핀치/휠 줌(최대 5배), 드래그 이동, 더블클릭(탭) 2배 토글, 우측 상단 X·ESC·배경 클릭으로 닫기 |
| **이미지 비율 자유화** | 16:9, 1:1 등 어떤 비율의 이미지도 잘리지 않고 전체 표시 (기존: 2:3 외 비율은 크롭됨) |
| **이전 스타일 저장** | "이전 스타일"로 이동할 때도 입력한 평가가 자동 저장 (기존: 앞으로 이동 시에만 저장 → 데이터 유실 가능성 제거) |
| **코디 없는 스타일 처리** | 코디 이미지가 없으면 "스타일링 선호도" 대신 "다음 스타일" 버튼 표시 — 불필요한 단계 건너뜀 |
| **코디평가 이전 단계 버튼** | 하단에 "< 이전 단계" 버튼 신설 — 상품정보로 복귀 동선 명확화 |
| **갤러리 개선** | 썸네일 카드에 품번 표시(좌상단), 종료 시 2단계 확인 팝업("앱을 종료하시겠습니까?" → "감사합니다."), 버튼 문구 정리(완료 / 시작하기) |
| **임직원1 상세정보** | 상품정보 진입 시 상세정보 자동 펼침 |
| **헤더 UI 통일** | 갤러리·상품정보·코디 상단 헤더의 항목 크기·세로 정렬·품번 표기 일관화 |

### **4. 관리자 기능 (Admin)**

- **양식 다운로드**: CSV(8컬럼) → **xlsx(20컬럼)** 로 확대 — `상품정보_업로드_양식.xlsx` (원단폭/단가/M-U/요척/원자재/부자재/공임/제조원가 등 상세정보 컬럼 포함)
- **상품정보 업로드**: **xlsx 파일 직접 업로드 지원** (기존 CSV도 계속 사용 가능) — 숫자·수식·하이퍼링크 셀 자동 변환, 빈 행 자동 제거
